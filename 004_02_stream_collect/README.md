# 004_02 — Stream.collect (Collectors deep dive)

This lesson drills into the Stream.collect terminal operation and the built-in `Collectors` helpers. It covers common collectors (toList, toSet, toMap), grouping/partitioning, downstream collectors, and practical tips for parallelism and immutability.

Prerequisites
- Java 17
- Maven 3.6+

Build
```bash
mvn -q package
```

Run example
```bash
mvn -q exec:java -Dexec.mainClass="dev.playground.lesson.Main"
```

See `lesson.md` for the full lesson content and exercises.
