# 003_01 Collections in Depth

This lesson covers Java Collections Framework in depth.

## How to Build and Run

### Prerequisites
- Java 17 or higher
- Maven 3.6 or higher

### Build the Project
```bash
mvn clean compile
```

### Run the Main Class
```bash
mvn exec:java -Dexec.mainClass="dev.playground.interview.Main"
```

### Run Tests
```bash
mvn test
```

### Package the Application
```bash
mvn package
```
